0. python 설치하기
https://www.python.org/downloads/release/python-379/
위 링크의 "Windows x86-64 executable installer" 파일을 다운 받아 설치합니다.
설치 시 Add .path 여부를 꼭!! 체크해주세요.
혹시 이미 파이썬 3.7이 설치되어있다면 건너뛸 수 있습니다.

1. Environment_Set.exe 실행하기
파이썬, 그 외 라이브러리 설치용 파일입니다.

2. 웹캠 설치하기
가능한 전신이 나올 수 있도록 설치해주세요.

3. Dance Game.exe 실행하기 
게임 파일로 혹시라도 정상작동하지 않는다면 알려주세요.
게임 맵 실행 시(게임 타이틀에서 플레이버튼 누르기) 
가능한 전신이 나오도록 캠에서 멀리 떨어져주세요!!

4. 종료 후 Environment_unSet.exe 실행하기
기존 설치한 라이브러리 제거용 파일입니다.
제거하지 않아도 문제는 없습니다.

5. python 제거하기
파이썬 설치 시 사용한 "python-3.7.9-amd64" 파일을 다시 실행하여 설치를 제거할 수 있습니다.
